# ととのうToDo

期限、タグ、依頼元・依頼先、添付ファイルをひとつに整理できる個人向けToDoツールです。PCとスマートフォンの両方で利用できます。

## 主な機能

- ToDo登録日時の自動記録
- 期限と状態（未着手・進行中・確認待ち・完了）の管理
- タグ、依頼元、依頼先、メモの登録
- ファイル添付と再ダウンロード
- 今日、期限超過、今後7日、完了済みの集計
- タイトル、タグ、依頼元・依頼先を横断する検索
- PC・スマートフォン向けのレスポンシブ表示

## データの保存場所

初版は個人利用向けです。ToDoと添付ファイルは、利用中のブラウザの保存領域（IndexedDB）に保存します。外部サーバーへの送信や、別端末との自動同期は行いません。

ブラウザの閲覧データを消去するとToDoも削除されるため、重要なファイルの原本は別途保管してください。
